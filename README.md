## **THE SILENT ATTACK**
IGI-Inspired FPS Game

## **Overview**
The IGI-Inspired FPS Game is a first-person shooter developed in Unity. It focuses on combat gameplay, player movement, and mission-based level design, providing an immersive FPS experience inspired by classic tactical shooters.

## **Features**
First-person movement and camera control  
Weapon shooting system  
Enemy combat interactions  
Level-based gameplay structure  
Interactive environment elements  
Smooth FPS controls and mechanics  

## **Technologies Used**
Game Engine: Unity  
Programming Language: C#  
Tools: Git, GitHub, Unity Editor, Visual Studio

git clone the repository

Open the project in Unity:
Open Unity Hub
Click on Open Project
Select the cloned folder

## **Run the game**
Open main scene in Unity
Click Play button

## **Usage**
Start the game from main menu or scene  
Use WASD for movement  
Use mouse for camera control  
Left click to shoot  
Complete level objectives by eliminating enemies
